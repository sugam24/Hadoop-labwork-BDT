# Lab Problems

This folder contains the Hadoop MapReduce lab materials and submissions.

Lab tasks overview (from the lab sheet):

- Task 1: Prepare a Linux environment and record system specs (CPU, memory) with terminal proof.
- Task 2: Install Hadoop 3.4.0, configure core-site.xml, hdfs-site.xml, mapred-site.xml, yarn-site.xml, and record a demo video of the setup and web UI.
- Task 3: Solve the two problems in this folder and record problem definition, input, and output in the video report.

- Problem1_WordCount: Word count on a plain text input file.
- Problem2_TitanicProblem: Mean age analysis on the Titanic dataset.

See implementation.md for detailed steps and notes.
